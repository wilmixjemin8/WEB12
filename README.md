WEB12
======

WEB12  is a  Framework  invented   by  wilmix jemin  j  at  year  2016  at  JAS.


WEB12   consists   of  modules    [WEB2  to  WEB12]


Note:  WEB12  is   used  only  with C/C++ P.L.


SYNTAX:
=======



<WEB2>

<WEB2@>

<%

<!   WEB2-web12  code>
%>

</WEB2>
